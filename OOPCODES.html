#01
Student Record System
EASY
classesencapsulation

Design a Student class to store a student's information and perform basic operations on it.

Requirements
  • Private data members: name (string), rollNo (int), gpa (float)
  • Public constructor to initialize all three attributes
  • Getter and setter methods for each attribute
  • A display() method that prints all student details in a formatted way
  • In main(), create two Student objects, set their data using setters, and display them
Use private for data and public for methods. Setters should validate that GPA is between 0.0 and 4.0.

Bank Account
EASY
classesencapsulation

Create a BankAccount class that simulates a simple bank account supporting deposits and withdrawals.

Requirements

  • Constructor to initialize holder name, account number, and an optional initial balance (default 0)
  • deposit(double amount) — add amount to balance; reject negative amounts
  • withdraw(double amount) — subtract if funds are sufficient; print error otherwise
  • getBalance() — return current balance
  • printStatement() — print all account details neatly
Default parameter syntax: BankAccount(string name, int num, double bal = 0.0). Guard against overdraft in withdraw.

Temperature Converter

#11
Fraction Class with Operator Concepts
EASY
classesmethods

Implement a Fraction class that stores a fraction and supports arithmetic operations as member methods.

Requirements
  • Private: numerator and denominator (both int)
  • Constructor: validate that denominator ≠ 0; throw or print error if it is
  • Private helper simplify() that reduces the fraction using GCD
  • Methods: add(Fraction), subtract(Fraction), multiply(Fraction), divide(Fraction) — each returns a new Fraction

#15
Stack Using a Class
